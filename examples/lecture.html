重构是 Vibe Coding 的第一等公民

Vibe Coding 不等于毫无章法的生成。它是"直觉式构建"与"结构化重构"的持续循环。没有重构兜底，Vibe 只是技术债的狂欢；掌握重构，Vibe 才是真正的超级生产力。

目标用户

有 1 年以上编码经验的开发者，使用过 AI 辅助编程工具，开始感到代码质量不可控。

学习目标

  • 识别 Vibe Coding 场景下代码腐化的典型信号
  • 掌握将「重构」嵌入编码循环的工作方式
  • 完成一次从混乱代码到可维护结构的实战推演

提纲

  1. 实战推演：用 Vibe Coding 写量潮课堂首页，得到一段能跑但混乱的代码
  2. 技术债分析——耦合、命名、边界遗漏
  3. 你有没有过这种感觉？——Vibe Coding 之后的隐秘不安
  4. "番茄小说"困境——能跑但混乱的代码是如何堆积的
  5. 为什么"功能实现"是一等公民？——可见性驱动的恶性循环
  6. 结构即思考——重构是将隐性理解显性化的过程
  7. 不只是代码——重构作为一种思维方式
时长 20 分钟 难度 中级 视频 + 可运行示例
