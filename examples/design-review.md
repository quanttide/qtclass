# 课时页面原型评审（设计师视角）

## 视觉层级扁平 ✅ 已解决

标题、描述、目标用户、学习目标、提纲、课时信息——所有 section 的视觉权重几乎一致（h2 统一 1.1rem，卡片内间距相同）。用户扫读时找不到视觉锚点。

已处理：标题区独立为 header（无 section 边框），h2 改为小号大写 + 品牌色，描述改为引号卡片样式，形成三层视觉阶梯。

## 信息分组缺少视觉节奏 ✅ 已解决

6 个 section 从上到下依次堆叠，中间用分隔线等距隔开，阅读节奏单一。

已处理：分为 3 个逻辑区——标题区（header，无边框）、内容区（3 个 section，border-top 分割）、元信息区（底部 tag 栏，品牌色浅底）。

## 课时信息视觉权重过轻 ✅ 已解决

课时信息放在底部独立 section，本身是辅助决策的关键信息，但当前样式（灰色小字、flex 排布）视觉权重最低。

已处理：改用品牌色 pill 标签 + 浅蓝底通栏，形成底部视觉落点。

## 字体系统未定义 ✅ 已解决

全文只有一种字号体系，缺少标题、正文、辅助信息的分层定义。描述段落未做行长控制（max-width 未应用于段落），宽屏下阅读体验差。

已处理：建立字号阶梯（h1 1.625rem → 正文 1rem → 引号 0.9375rem → 标签 0.8125rem → section 标题 0.75rem），段落添加 max-width: 36em 控制行长。

## 缺乏品牌属性 ⚠️ 部分解决

当前是通用卡片样式，没有量潮课堂的品牌色、品牌字体、图标系统。

已补充 CSS 变量体系（brand / text / bg 色板），品牌色 #2563eb 贯穿页面。完整品牌系统需量潮课堂设计规范落地后补充（字体、图标、Logo、间距 scale）。
